Portfolio

Goal-E

Goal-E is a full-stack application developed for people with aspirations. Users create an account and add goals they'd like to accomplish and along with the steps to do so. The user may update goals and steps as needed, create multiple goals, and delete goals. Goal-E was developed using mySQL, express, and express-handlebars.

Side Beat

Developed to connect concert-seekers with shows in their area based on their favorite artist or band, Side Beat uses two 3rd-party APIs -- Taste Dive and SeatGeek. Users enter an artist or band name they already love and a list of similar artists or bands is displayed with links to buy consert tickets.

Day Planner

A calendar app using jQuery to dynamically power HTML and CSS. Time blocks are color-coded to show past, present, and future. The text input is saved in local storage and persists upon refreshing the page.

Coding Quiz

A multiple choice quiz application that tests users' coding knowledge in HTML, CSS, and JavaScript. Upon completion of the quiz, users have the opportunity to save their initials and score to local storage.

Horiseon Code Refactor

An example of code refactoring that includes adding the company name for browser title, fixing the navbar by adding an id, and cosolidating redundant lines in CSS that share the same styling.
